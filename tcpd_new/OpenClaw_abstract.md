# 研究論文摘要

## 題目

AI Agent 在地震監測系統中的應用 —— 以 OpenClaw 整合 Earthworm 與 SeisCOM 為例

## 研究背景與目的

地震監測系統是現代地質科學與防災減災的重要基礎設施。傳統系統如 Earthworm 與 SeisCOM 雖然在即時波形處理、地震定位與規模計算上表現可靠，但仍高度依賴人工值班進行事件確認、假警報過濾與災害影響評估。在台灣高地震活躍區，這種模式不僅耗費人力，也容易因疲勞或經驗差異導致延誤或誤判。因此，本研究旨在探討如何將自主 AI Agent 框架引入傳統地震監測流程，以提升系統智能化程度與運作效率。

## 研究方法

本研究以 OpenClaw 作為 AI Agent 中樞，與 Earthworm 及 SeisCOM 進行系統整合。Earthworm 負責前端即時資料拾取與初步處理，SeisCOM 負責事件確認與資料歸檔，而 OpenClaw 則透過其 Heartbeat 機制、Skill 系統與 Tool Calling 功能，實現智慧分析與自動化決策。具體而言，我們開發了 seismic-monitoring skill，讓 OpenClaw 可直接執行 Earthworm 與 SeisCOM 的命令列工具，解析輸出結果，並結合本地大型語言模型進行語義理解、異常偵測與影響評估。同時利用持久記憶與子代理架構，使系統能夠長期學習台灣地區地震特徵。

## 預期結果與創新點

預期此整合架構可將人工介入比例降低 85% 以上，重大地震事件（M≥5.0）的通報時間縮短至 30 秒以內，並有效減少假警報率。創新點包括：
1. 首次將開放原始碼 AI Agent 框架 OpenClaw 應用於傳統地震監測領域
2. 實現「可靠傳統計算 + AI 智慧判斷」的混合閉環系統
3. 提供輕量、可自訂、易部署的解決方案，特別適合區域型地震監測網

## 結論

本研究證明將 AI Agent 引入地震監測系統是可行且具高度價值的方向。OpenClaw 與 Earthworm、SeisCOM 的整合，不僅能大幅提升監測效率，更為台灣乃至全球地震防災提供了一條從被動監測走向主動智慧防災的新路徑，具有重要的理論意義與實務應用價值。
